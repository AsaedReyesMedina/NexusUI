# Guía de Instalación

## Requisitos Previos

Antes de comenzar con la instalación, asegúrate de tener instalado:

- Node.js (versión 18 o superior)
- Yarn (gestor de paquetes)
- Git

## Instalación

1. Clona el repositorio:
```bash
git clone [URL_DEL_REPOSITORIO]
cd NexusUI
```

2. Instala las dependencias:
```bash
yarn install
```

## Scripts Disponibles

El proyecto incluye varios scripts útiles:

- `yarn build`: Compila el proyecto para producción
- `yarn dev`: Inicia el modo de desarrollo con recarga automática
- `yarn mf-dev`: Inicia el servidor de desarrollo con Module Federation
- `yarn storybook`: Inicia Storybook en modo desarrollo
- `yarn build-storybook`: Construye la versión estática de Storybook

## Estructura del Proyecto

```
NexusUI/
├── src/              # Código fuente
├── .storybook/       # Configuración de Storybook
├── dist/             # Archivos compilados
└── node_modules/     # Dependencias
```

## Configuración

El proyecto utiliza:
- TypeScript para el tipado estático
- React 19 como framework principal
- TailwindCSS para estilos
- Storybook para documentación y desarrollo de componentes
- Module Federation para arquitectura de micro-frontends

## Desarrollo

Para comenzar a desarrollar:

1. Inicia el servidor de desarrollo:
```bash
yarn dev
```

2. Para ver los componentes en Storybook:
```bash
yarn storybook
```

## Construcción para Producción

Para construir el proyecto para producción:

```bash
yarn build
```

Los archivos compilados se generarán en el directorio `dist/`.

## Soporte

Si encuentras algún problema durante la instalación o el desarrollo, por favor:

1. Revisa la documentación existente
2. Verifica que cumples con todos los requisitos previos
3. Asegúrate de tener las versiones correctas de las dependencias
